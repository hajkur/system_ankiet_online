# Aplikacja Ankiet online

Aplikacja zaliczeniowa na PPK Programowanie zaawansowane.

Aplikacja ma umożliwić tworzenie ankiet, oraz głosowanie w niej dla użytkowników.

## Wymagania

- .NET 9.0 SDK

## Jak uruchomić

w folderze z plikami w terminalu trzeba wpisać 

'dotnet run'

Aplikacja automatycznie utworzy konto z uprawnieniami administratora o danych:

admin@gmail.com | Admin123!

Strona będzie dostępna pod adresami:
- HTTP: `http://localhost:5296`
- HTTPS: `https://localhost:7215`

Przy połączeniu HTTPS może wymagać potwierdzenia niebezpiecznej strony.
